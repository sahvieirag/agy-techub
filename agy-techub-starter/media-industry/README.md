# Gemini Enter-View - Omnichannel Content Explosion Engine 🎙️💥

Este projeto é uma demonstração interativa e premium desenvolvida para o setor de **Mídia, Entretenimento & Marketing (Media)**, ilustrando o poder da suíte **Gemini Enter-View** em orquestrar a geração autônoma de ativos multiplataforma a partir de um único insumo textual de áudio transcrito.

## ✨ Funcionalidades Demonstradas

* **Upload e Transcrição Base:** Simula o upload de arquivos de podcast em áudio e exibe a transcrição para processamento.
* **Geração de Conteúdo Omnichannel:** Criação simultânea de:
  * Um artigo de blog de SEO estruturado.
  * Um post interativo focado no feed corporativo do LinkedIn.
  * Uma thread encadeada e dinâmica para o X (antigo Twitter).
  * Uma newsletter profissional formatada para e-mail.
* **Carrossel Criativo com Imagens Reais por IA:** Um carrossel interativo exibindo banners profissionais gerados previamente por IA (`banner-ai.png` e `podcast-studio.png`) sem usar simples placeholders de texto.
* **Design System Ad-Tech Elegante:** Layout de 3 colunas em **Tema Claro** com efeitos de glassmorphism em tons lavanda, abas animadas de rede social realistas e consoles interativos.

## 🛠️ Como Executar Localmente

### Pré-requisitos
* **Python 3** instalado em sua máquina.

### Passos para rodar
1. Certifique-se de que está na pasta raiz do repositório clonado do workshop:
   ```bash
   cd agy-techub
   ```
2. Inicialize o nosso servidor local de portal:
   ```bash
   python3 -m http.server 8090
   ```
3. Abra o seu navegador e acesse diretamente: **`http://localhost:8090/agy-techub-starter/media-industry/index.html`** 💻
